### 架构：切换流程

```
启动时
  └─ discovery.ScanWithModel() → 发现本地引擎
  └─ 选择 fallback 候选（优先 OpenAI-compatible）
  └─ 启动 monitorFailover goroutine（每 30s）

监控循环
  └─ checkAPIReachable() + checkEngineReachable()
  └─ state.Decide(apiOK, localOK)  // 带滞后防抖
  └─ 切换到 local:
       ├─ llm.SetProvider("local")
       ├─ k.Router.SetStrategy(LocalRouteStrategy)
       └─ os.Setenv("LLM_BASE_URL", engine.Endpoint)
  └─ 切回 api（滞后 2 次成功）:
       ├─ llm.SetProvider("")    // 恢复默认
       ├─ k.Router.SetStrategy(nil)
       └─ os.Unsetenv("LLM_BASE_URL")
```

### 关键设计决策

- **LocalRouteStrategy 持有 Router 引用** — 复用 buildPluginList() + parseDecision()，硬化层校验完全一致
- **线程安全 SetProvider** — RWMutex 保护 overrideProvider，不影响活跃请求
- **OpenAI-compatible 优先** — selectFallbackEngine 优先选 openai/llamacpp 类型（支持 /v1/chat/completions）
- **LLM_BASE_URL/LLM_MODEL 环境变量覆盖** — 切换时设/清环境变量，ChatCompletionWithUsage 自动适配
