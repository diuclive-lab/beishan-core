# 开发日志 2026-05-23

## code_deep_analyze 工作流改造：去本地模型化

### 背景

`code_deep_analyze.yaml` 的 `analyze_files` 步骤原使用 `provider: local`（Qwen3.6 本地模型）做逐文件分析。本地模型速度慢且需要 GPU。

### 改动

将 `analyze_files` 步骤的 `provider` 从 `local` 改为 `deepseek`，全部走 API：

| 位置 | 改前 | 改后 |
|------|------|------|
| provider | `local` | `deepseek` |
| 成本估算 | DeepSeek 2次 + Qwen 免费 | DeepSeek 3次 ≈ ¥0.08 |
| 阶段标题 | Qwen3.6 并发批量逐文件深读 | DeepSeek 逐文件分析 |
| 标签 | 双模型分析 | 全 API 分析 |

### 验证

`go build ./...` 通过后，用 beishan-core 自身的 `code_deep_analyze` 工作流做了端到端测试。8 步全部成功，`analyze_files` 改用 DeepSeek API 后输出正确 JSON 格式分析结果。
